## Clanker

Use [crush](https://charm.land/crush/) with the `GLM-5.1` model ([hyper](https://hyper.charm.land)).

1. add an edge case as test and commit it
2. tell the clanker to `fix test`

> It can handle a small amount amount of new tests pretty well, but stumbles a bit on larger changes.
> Best to iterate in small steps of ~1-5 modifications.
